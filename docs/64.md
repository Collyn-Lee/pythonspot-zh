## Django

如果要开始 python Web 开发，可以使用一个名为 Django 的 Web 框架。它被设计为快速，安全和可扩展的。 它带有一个对象关系映射器（ORM），这意味着 Python 中的对象被映射到数据库中的对象。

用 Django 创建的应用程序分为三个独立的层：模型（数据库），视图（外观）和控制器（逻辑），或者简称为模型-视图-控制器（MVC）体系结构。
