## 运行 Python 代码

A python program should be save as a file with a .py extension.
Try this code:

```py
print("Hello World!")
print("This is a Python program.")

```

预期产量：

```py
Hello World!
This is a Python program

```

如果使用解释器，请使用：

```py
python program.py

```
